Analyze Network Artifact
Use this for manual packet text, logs, protocol output, or suspicious observations.
OR
Supported: .pcap, .pcapng, .txt, .log, .csv, .json
How It Works
  • Accepts text or uploaded network artifacts.
  • Uses PyShark for PCAP parsing when available.
  • Falls back to Scapy when needed.
  • Scores risk as Low / Medium / High / Critical.
  • Stores results in local SQLite history.
  • Writes SHA256 evidence hash to local chain ledger.
  • Supports local PDF report export.
Alert Simulation

All alerts stay local inside the application using on-screen notifications. No external email, SMS, or cloud dependency is used.
