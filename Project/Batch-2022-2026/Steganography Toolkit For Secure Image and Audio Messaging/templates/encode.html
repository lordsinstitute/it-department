Encode into Carrier
Upload your carrier after you’re satisfied with the recommendation.
Supported: PNG, BMP, WAV (PCM). For WAV, use redundancy carefully (capacity shrinks by R).
Enterprise-grade Encoding Details
  • Batch container stores multiple labeled records.
  • Compression enabled only if it reduces size.
  • Encryption via PBKDF2 + stream XOR; HMAC tag for authenticity.
  • Watermark ID embedded into header and written to ledger evidence.
  • Adaptive embedding prefers edges (images).
  • Redundancy repeats bits; extraction could majority-vote later.
Rule of thumb
Keep usage < 30%. If you need more, choose a bigger carrier, or reduce redundancy, or use compression.
