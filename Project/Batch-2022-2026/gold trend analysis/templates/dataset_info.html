📊 Dataset Information

Complete details about the gold price dataset

📁 Dataset Overview

  • Dataset Name: gold_data.csv
  • Total Records: 730 days (2 years)
  • Total Rows: 730 rows
  • Total Columns: 6 columns (Date, Open, High, Low, Close, Volume)
  • Data Frequency: Daily
  • Date Range: Last 2 years from current date
  • File Format: CSV (Comma-Separated Values)
  • Data Source: Historical gold price data
  • Dataset Shape: (730, 6) - 730 rows × 6 columns

Indicator Full Name Window/Period Purpose

🔧 Final Features for Training

  • Feature 1: Close Price (Target variable)
  • Feature 2: Volume (Trading activity)
  • Feature 3: MA_10 (Short-term trend)
  • Feature 4: MA_30 (Long-term trend)
  • Feature 5: RSI (Momentum indicator)
  • Feature 6: MACD (Trend strength)

📌 Total Features: 6 features per time step
📌 Sequence Length: 60 days
📌 Input Shape: (60, 6)

⚙️ Data Preprocessing Steps

  • Step 1: Load CSV with Date as index
  • Step 2: Remove null values (dropna)
  • Step 3: Calculate technical indicators
  • Step 4: Normalize using MinMaxScaler (0-1)
  • Step 5: Create 60-day sequences
  • Step 6: Split 80% train, 20% test
