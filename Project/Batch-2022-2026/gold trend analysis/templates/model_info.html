🧠 Model Architecture & Mathematics

Deep dive into LSTM model, algorithms, and mathematical formulas

🏗️ Model Architecture Overview

  • Model Type: LSTM (Long Short-Term Memory)
  • Framework: TensorFlow/Keras 2.8.0
  • Model Class: Sequential Neural Network
  • Total Layers: 6 layers (2 LSTM + 2 Dropout + 2 Dense)
  • Input Shape: (60, 6) - 60 time steps × 6 features
  • Output: Single value (next day's closing price)

📐 Layer-by-Layer Architecture

Layer 1: LSTM Layer (First)

  • Units: 50 neurons
  • Return Sequences: True (passes sequences to next LSTM)
  • Input Shape: (60, 6)
  • Output Shape: (60, 50)
  • Parameters: ~11,400

Layer 2: Dropout Layer (First)

  • Dropout Rate: 0.2 (20%)
  • Purpose: Prevent overfitting
  • Randomly drops: 10 out of 50 neurons during training

Layer 3: LSTM Layer (Second)

  • Units: 50 neurons
  • Return Sequences: False (outputs single vector)
  • Input Shape: (60, 50)
  • Output Shape: (50,)
  • Parameters: ~20,200

Layer 4: Dropout Layer (Second)

  • Dropout Rate: 0.2 (20%)
  • Purpose: Additional regularization

Layer 5: Dense Layer (Hidden)

  • Units: 25 neurons
  • Activation: Linear (default)
  • Input Shape: (50,)
  • Output Shape: (25,)
  • Parameters: 1,275

Layer 6: Dense Layer (Output)

  • Units: 1 neuron
  • Activation: Linear
  • Input Shape: (25,)
  • Output Shape: (1,)
  • Parameters: 26
  • Output: Predicted closing price (normalized)

🔢 Mathematical Formulas

1. LSTM Cell Equations

Forget Gate: ft = σ(Wf · [ht-1, xt] + bf) Input Gate: it = σ(Wi · [ht-1, xt] + bi) Candidate Cell State: C̃t = tanh(WC · [ht-1, xt] + bC) Cell State Update: Ct = ft * Ct-1 + it * C̃t Output Gate: ot = σ(Wo · [ht-1, xt] + bo) Hidden State: ht = ot * tanh(Ct) Where: - σ = Sigmoid activation (0 to 1) - tanh = Hyperbolic tangent (-1 to 1) - W = Weight matrices - b = Bias vectors - * = Element-wise multiplication

2. Loss Function (Mean Squared Error)

MSE = (1/n) × Σ(yactual - ypredicted)² Where: - n = Number of samples - yactual = True closing price - ypredicted = Model's predicted price

3. Adam Optimizer Update Rule

mt = β₁ × mt-1 + (1 - β₁) × gt vt = β₂ × vt-1 + (1 - β₂) × gt² m̂t = mt / (1 - β₁t) v̂t = vt / (1 - β₂t) θt = θt-1 - α × m̂t / (√v̂t + ε) Where: - β₁ = 0.9 (first moment decay) - β₂ = 0.999 (second moment decay) - α = Learning rate - ε = 1e-8 (small constant) - gt = Gradient at time t

4. MinMax Normalization

Xscaled = (X - Xmin) / (Xmax - Xmin) Inverse Transform: Xoriginal = Xscaled × (Xmax - Xmin) + Xmin

5. Technical Indicators Formulas

Moving Average (MA): MAn = (P₁ + P₂ + ... + Pn) / n RSI (Relative Strength Index): RS = Average Gain / Average Loss RSI = 100 - (100 / (1 + RS)) MACD: MACD = EMA12 - EMA26 EMA (Exponential Moving Average): EMAt = (Pt × k) + (EMAt-1 × (1 - k)) where k = 2 / (n + 1)

🎯 Model Training Process

  • Step 1: Load 730 days of gold price data
  • Step 2: Calculate 4 technical indicators (MA10, MA30, RSI, MACD)
  • Step 3: Normalize all 6 features to 0-1 range
  • Step 4: Create sequences of 60 consecutive days
  • Step 5: Split into 584 training + 146 test samples
  • Step 6: Build LSTM model with 6 layers
  • Step 7: Train for 30-50 epochs with batch size 32
  • Step 8: Validate on test set after each epoch
  • Step 9: Save trained model as gold_model.h5
  • Step 10: Save scaler as scaler.pkl for predictions

🔮 Prediction Process

  • Step 1: Load trained model (gold_model.h5)
  • Step 2: Fetch last 60 days of data
  • Step 3: Calculate technical indicators
  • Step 4: Normalize using saved scaler
  • Step 5: Reshape to (1, 60, 6) for LSTM input
  • Step 6: Model predicts normalized value
  • Step 7: Inverse transform to get actual price
  • Step 8: Calculate direction (UP/DOWN) and change %
  • Step 9: Return prediction with confidence score

💡 Key Algorithms Used

  • LSTM (Long Short-Term Memory): Handles sequential time-series data with memory cells
  • Adam Optimizer: Adaptive learning rate optimization algorithm
  • Dropout Regularization: Randomly drops neurons to prevent overfitting
  • MinMax Scaling: Normalizes features to 0-1 range
  • Backpropagation Through Time (BPTT): Trains recurrent networks
  • Gradient Descent: Minimizes loss function iteratively
